This microservice functions as a URL shortener.

Example usage:

Send one parameter with the url to be shortened, in the following format :

https://urli.herokuapp.com/new/?url=https://www.youtube.com/watch?v=sYZXogw8AtA

Example result:

{"inputUrl":"https://www.youtube.com/watch?v=sYZXogw8AtA","shortUrl":"urli.herokuapp.com/d"}
